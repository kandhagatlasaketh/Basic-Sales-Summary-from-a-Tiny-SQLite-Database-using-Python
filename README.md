 Basic Sales Summary Using SQLite & Python

##  Objective  
To connect Python with a SQLite database, run basic SQL queries to summarize sales data (like total quantity and total revenue by product), and visualize the result using a bar chart.

---

##  Tools & Technologies  
| Tool | Purpose |
|------|---------|
| Python | Main programming language |
| sqlite3 | Connect and query SQLite database |
| pandas | Load SQL results into DataFrame |
| matplotlib | Create bar chart visualization |
| SQLite (sales_data.db) | Small local database |

Learning Outcomes

By completing this task, I learned:
How to connect Python with a SQLite database
How to write basic SQL queries inside Python
How to load SQL results into pandas
How to perform summary calculations (total & revenue)
How to visualize data using matplotlib.
